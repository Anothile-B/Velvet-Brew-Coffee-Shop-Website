# Velvet Brew Coffee Shop Website
---

# Project Overview
This project is a responsive front-end website built to represent a stylish coffee shop. 
It focuses on clean design, smooth user experience, and visually appealing layout to attract customers.

---

# Live Demo Link
- GitHub Pages Link: https://anothile-b.github.io/Velvet-Brew-Coffee-Shop-Website/ 
---

# Features
- Beautiful homepage with branding and hero section
- Menu section showcasing coffee and beverages
- Image sections to highlight the café atmosphere
- Responsive design for mobile and desktop
- Smooth and modern UI using CSS styling
- Light interactivity using JavaScript
- Contact form and About page

---

# Technologies Used
- HTML5: Structure of the website
- CSS3:  Styling and layout design
- JavaScript: Basic interactivity

---

# Purpose
This project was created to:
- Practice front-end web development
- Improve HTML and CSS design skills
- Build a visually appealing website
- Understand basic JavaScript interactions
